# Задание 13. Трассировка лучей (Ray Tracing)

## Описание

Рендеринг сцены методом трассировки лучей: пересечение лучей с объектами, тени, отражения, преломления. Результат сохраняется в `output.png`.

## Теория

### Уравнение луча

$$P(t) = O + t \cdot D, \quad t > 0$$

где $O$ — начало луча, $D$ — направление.

### Пересечение со сферой

Сфера с центром $C$ и радиусом $r$: $|P(t) - C|^2 = r^2$. Подставляя луч:

$$a t^2 + b t + c = 0$$

$$a = D \cdot D, \quad b = 2 \, D \cdot (O - C), \quad c = (O - C) \cdot (O - C) - r^2$$

$$\Delta = b^2 - 4ac$$

Если $\Delta \geq 0$: $\; t = \frac{-b \pm \sqrt{\Delta}}{2a}$. Берём наименьшее положительное $t$.

### Пересечение с плоскостью

Для горизонтальной плоскости $y = k$:

$$t = \frac{k - O_y}{D_y}$$

### Нормаль сферы

$$\vec{N} = \text{normalize}(P - C)$$

### Тени

Из точки пересечения $P$ пускается луч к источнику света. Если луч пересекает другой объект — точка в тени.

### Отражение

$$\vec{R} = \vec{D} - 2(\vec{D} \cdot \vec{N}) \, \vec{N}$$

### Преломление (закон Снеллиуса)

$$\frac{\sin\theta_2}{\sin\theta_1} = \frac{n_1}{n_2}$$

Вектор преломлённого луча при $\eta = n_1 / n_2$:

$$\vec{T} = \eta \, \vec{D} + (\eta \cos\theta_1 - \cos\theta_2) \, \vec{N}$$

где $\cos\theta_1 = -\vec{D} \cdot \vec{N}$, $\;\cos\theta_2 = \sqrt{1 - \eta^2 (1 - \cos^2\theta_1)}$.

Если выражение под корнем отрицательно — полное внутреннее отражение.

### Коэффициент Френеля (упрощённый)

Определяет соотношение отражённой и преломлённой составляющих в зависимости от угла падения.

## Алгоритм

1. Для каждого пикселя экрана сформировать первичный луч из камеры.
2. Найти ближайшее пересечение луча с объектами сцены.
3. Вычислить освещение (Lambert), проверить тень.
4. Рекурсивно трассировать отражённый / преломлённый луч.
5. Сложить вклады и записать цвет пикселя.
6. Сохранить результат в `output.png`.

## Варианты

| Вариант | Описание |
|---------|----------|
| **А** (`a.c`) | Непрозрачные сферы. Первичные лучи + тени. Экспорт в PNG. |
| **Б** (`b.c`) | + отражения (1 bounce). Плоскость-пол с шахматной текстурой. |
| **В** (`c.c`) | + преломление (Снеллиус). Три сферы: прозрачная, зеркальная, матовая. |

## Управление

- Программа запускается, рендерит сцену и сохраняет результат в `output.png`.
- Интерактивное управление не предусмотрено (офлайн-рендер).

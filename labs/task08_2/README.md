# Задание 8.2. Коррекция цвета

## Описание

Реализация базовых операций цветокоррекции изображений в пространствах RGB и HSV.

## Теория

### Яркость и контраст (RGB)

$$\text{new} = \text{clamp}\bigl((\text{pixel} - 128) \cdot \text{contrast} + 128 + \text{brightness},\; 0,\; 255\bigr)$$

### Цветовая модель HSV

- $H$ — тон ($0°$–$360°$), $S$ — насыщенность ($0$–$1$), $V$ — яркость ($0$–$1$)

**RGB → HSV:**

$$V = \max(R, G, B), \quad S = \frac{V - \min(R, G, B)}{V}$$

$H$ определяется по максимальному каналу.

**HSV → RGB:**

$$c = V \cdot S, \quad x = c \left(1 - \left| \frac{H}{60} \bmod 2 - 1 \right|\right), \quad m = V - c$$

### Сепия

$$R' = 0.393R + 0.769G + 0.189B$$
$$G' = 0.349R + 0.686G + 0.168B$$
$$B' = 0.272R + 0.534G + 0.131B$$

### Негатив

$$R' = 255 - R, \quad G' = 255 - G, \quad B' = 255 - B$$

### Гамма-коррекция

$$\text{out} = 255 \cdot \left(\frac{\text{in}}{255}\right)^\gamma$$

## Алгоритм

1. Загрузить изображение
2. Для каждого пикселя — применить выбранное преобразование
3. Ограничить значения каналов диапазоном $[0, 255]$
4. Отобразить результат

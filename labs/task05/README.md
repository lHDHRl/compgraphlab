# Задание 5. Триангуляция полигона

## Описание

Разбиение полигона на треугольники: веерная триангуляция, алгоритм ear clipping, обработка полигонов с отверстиями.

## Теория

### Векторное произведение (cross product)

Для точек $O$, $A$, $B$:

$$\text{cross}(O, A, B) = (A_x - O_x)(B_y - O_y) - (A_y - O_y)(B_x - O_x)$$

- $\text{cross} > 0$ — поворот против часовой стрелки (выпуклый угол при CCW-обходе)
- $\text{cross} < 0$ — поворот по часовой стрелке (рефлексный угол)

### Определение ориентации полигона

Знаковая площадь:

$$S = \frac{1}{2} \sum_{i=0}^{N-1} (x_i y_{i+1} - x_{i+1} y_i)$$

$S > 0$ — обход против часовой стрелки (CCW), $S < 0$ — по часовой (CW).

### Веерная триангуляция

Для выпуклого полигона с вершинами $v_0, v_1, \ldots, v_{N-1}$:

$$\text{треугольники: } (v_0, v_i, v_{i+1}), \quad i = 1, \ldots, N-2$$

Всего $N - 2$ треугольников.

### Алгоритм Ear Clipping

Вершина $v_i$ является «ухом», если:

1. Треугольник $(v_{i-1}, v_i, v_{i+1})$ выпуклый:
$$\text{cross}(v_{i-1}, v_i, v_{i+1}) > 0 \quad \text{(для CCW-полигона)}$$

2. Ни одна другая вершина полигона не лежит внутри треугольника $(v_{i-1}, v_i, v_{i+1})$.

**Проверка точки $P$ внутри треугольника $(A, B, C)$:**

$$d_1 = \text{cross}(A, B, P), \quad d_2 = \text{cross}(B, C, P), \quad d_3 = \text{cross}(C, A, P)$$

Точка внутри, если $d_1$, $d_2$, $d_3$ одного знака.

Алгоритм: найти ухо → отрезать → повторить, пока не останется 3 вершины.

### Полигон с отверстием (мостик)

1. Найти ближайшие вершины внешнего ($v_{\text{out}}$) и внутреннего ($v_{\text{in}}$) контуров
2. Построить мостик: соединить $v_{\text{out}} \to v_{\text{in}}$, обойти внутренний контур, вернуться $v_{\text{in}} \to v_{\text{out}}$
3. Получить единый замкнутый контур (мостик проходится дважды: туда и обратно)
4. Применить ear clipping к объединённому контуру

## Алгоритм работы

1. Задать полигон (из задания 4 или вручную)
2. Определить ориентацию, при необходимости — инвертировать
3. Выполнить триангуляцию
4. Отобразить треугольники разными цветами

## Варианты

### Вариант А (`a.c`)
Веерная триангуляция выпуклого полигона ($N = 13$ вершин). Все треугольники строятся от вершины $v_0$.

### Вариант Б (`b.c`)
Ear clipping для звёздчатого (невыпуклого) полигона. Рефлексные вершины пропускаются при поиске ушей.

### Вариант В (`c.c`)
Полигон с отверстием. Построение мостика между внешним и внутренним контурами, затем ear clipping объединённого контура.

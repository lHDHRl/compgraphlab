# Задание 4. Генерация полигона

## Описание

Генерация различных типов полигонов: выпуклый, звёздчатый, самопересекающийся. $N = 13$ вершин.

## Теория

### Выпуклый полигон на эллипсе

$N$ точек равномерно распределены по эллипсу:

$$x_i = c_x + r_x \cos\frac{2\pi i}{N}, \quad y_i = c_y + r_y \sin\frac{2\pi i}{N}, \quad i = 0, \ldots, N-1$$

### Звёздчатый полигон

Чередование внешнего ($R$) и внутреннего ($r$) радиусов. Всего $2N$ вершин:

$$x_i = c_x + \rho_i \cos\frac{2\pi i}{2N}, \quad y_i = c_y + \rho_i \sin\frac{2\pi i}{2N}$$

где

$$\rho_i = \begin{cases} R, & i \text{ чётное} \\ r, & i \text{ нечётное} \end{cases}$$

### Самопересекающийся полигон (бабочка)

Полярная кривая:

$$r(\theta) = \sin(2\theta) + 0.5 \sin(3\theta)$$

Декартовы координаты:

$$x_i = r(\theta_i) \cos\theta_i, \quad y_i = r(\theta_i) \sin\theta_i, \quad \theta_i = \frac{2\pi i}{N}$$

## Алгоритм работы

1. Вычислить координаты $N$ вершин по соответствующей формуле
2. Соединить вершины последовательно, замкнуть контур
3. При необходимости — выполнить заливку

## Управление

| Клавиша | Описание |
|---|---|
| `+` / `-` | Изменение количества вершин |
| `F` | Переключение заливка / контур |
| `ESC` | Выход |

## Варианты

### Вариант А (`a.c`)
Выпуклый полигон на эллипсе ($N = 13$). Отображается заполненным и с контуром.

### Вариант Б (`b.c`)
Звёздчатый полигон: 26 вершин (13 внешних на радиусе $R$, 13 внутренних на радиусе $r$).

### Вариант В (`c.c`)
Самопересекающийся полигон по полярной кривой $r(\theta) = \sin 2\theta + 0.5 \sin 3\theta$ (бабочка).

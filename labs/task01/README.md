# Задание 1. Поворот отрезка

## Описание

Реализация поворота отрезков и фигур на плоскости с использованием алгоритмов растеризации Брезенхема и Ву.

## Теория

### Матрица поворота 2D

Поворот точки $(x, y)$ на угол $\alpha$ вокруг начала координат:

$$x' = x \cos\alpha - y \sin\alpha$$
$$y' = x \sin\alpha + y \cos\alpha$$

### Поворот вокруг произвольной точки $(c_x, c_y)$

1. Сдвиг к началу: $x_1 = x - c_x$, $y_1 = y - c_y$
2. Поворот: $x_2 = x_1 \cos\alpha - y_1 \sin\alpha$, $y_2 = x_1 \sin\alpha + y_1 \cos\alpha$
3. Сдвиг обратно: $x' = x_2 + c_x$, $y' = y_2 + c_y$

### Алгоритм Брезенхема

Растеризация отрезка целочисленной арифметикой. Для отрезка от $(x_0, y_0)$ до $(x_1, y_1)$:

- $dx = |x_1 - x_0|$, $dy = |y_1 - y_0|$
- Ошибка накопления: $e = 2 \cdot dy - dx$
- На каждом шаге: если $e > 0$, сдвиг по $y$, $e \mathrel{-}= 2 \cdot dx$; всегда сдвиг по $x$, $e \mathrel{+}= 2 \cdot dy$

### Алгоритм Ву (антиалиасинг)

Для каждой точки отрезка вычисляется дробная часть координаты $f$. Два соседних пикселя закрашиваются с интенсивностями:

$$I_1 = 1 - f, \quad I_2 = f$$

где $f = y - \lfloor y \rfloor$ — дробная часть текущей координаты.

## Алгоритм работы

1. Задать координаты отрезка (или фигуры)
2. Вычислить новые координаты вершин по формуле поворота
3. Растеризовать отрезки алгоритмом Брезенхема (или Ву)
4. Отобразить результат

## Управление

| Клавиша/действие | Описание |
|---|---|
| `LEFT` / `RIGHT` | Поворот на фиксированный угол |
| `SPACE` | Старт/стоп автовращения (вариант В) |
| Мышь (ЛКМ) | Задать центр поворота (вариант Б) |
| `ESC` | Выход |

## Варианты

### Вариант А (`a.c`)
Один отрезок, координаты вводятся с клавиатуры. Растеризация алгоритмом Брезенхема. Поворот клавишами `LEFT`/`RIGHT`.

### Вариант Б (`b.c`)
Контур треугольника. Центр поворота задаётся мышью. Растеризация алгоритмом Брезенхема.

### Вариант В (`c.c`)
Звезда из 10 вершин. Автоматическое вращение по таймеру (`SPACE` — старт/стоп). Антиалиасинг алгоритмом Ву.

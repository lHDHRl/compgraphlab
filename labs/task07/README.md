# Задание 7. Закраска (Shading)

## Описание

Реализация моделей освещения и методов закраски трёхмерных объектов.

## Теория

### Модель освещения Фонга

$$I = I_a k_a + I_d k_d \max(0,\; \vec{N} \cdot \vec{L}) + I_s k_s \max(0,\; \vec{R} \cdot \vec{V})^n$$

- $I_a, I_d, I_s$ — интенсивности фонового, диффузного и зеркального света
- $k_a, k_d, k_s$ — коэффициенты материала
- $\vec{N}$ — нормаль поверхности
- $\vec{L}$ — направление к источнику света
- $\vec{V}$ — направление к камере
- $n$ — показатель блеска (shininess)

Отражённый вектор:

$$\vec{R} = 2(\vec{N} \cdot \vec{L})\vec{N} - \vec{L}$$

### Закраска Гуро

1. Вычислить освещение $I$ в каждой вершине треугольника
2. Интерполировать интенсивность по площади треугольника через барицентрические координаты:

$$I_p = \lambda_1 I_1 + \lambda_2 I_2 + \lambda_3 I_3$$

### Закраска Фонга

1. Интерполировать нормали по треугольнику:

$$\vec{N}_p = \text{normalize}(\lambda_1 \vec{N}_1 + \lambda_2 \vec{N}_2 + \lambda_3 \vec{N}_3)$$

2. Вычислить освещение попиксельно по интерполированной нормали

### Нормали

- **Сфера** в точке $P$: $\vec{N} = \text{normalize}(P - C)$, где $C$ — центр
- **Тор**: через частные производные параметрической поверхности по $u$ и $v$

### Теневая матрица

Проецирование объекта на плоскость $y = \text{const}$ вдоль направления к источнику света.

## Алгоритм

1. Построить полигональную модель (сфера, тор)
2. Вычислить нормали в вершинах
3. Для каждого пикселя треугольника — определить барицентрические координаты
4. Применить Гуро (интерполяция интенсивности) или Фонг (интерполяция нормалей)

## Варианты

### Вариант А (`a.c`)

Сфера $16 \times 16$ сегментов, один источник света, закраска Гуро.

| Клавиша | Действие |
|---------|----------|
| ← / → / ↑ / ↓ | Перемещение источника света |

### Вариант Б (`b.c`)

Сфера + тор, переключение метода закраски, фоновое освещение.

| Клавиша | Действие |
|---------|----------|
| TAB | Переключение Гуро / Фонг |
| ← / → / ↑ / ↓ | Перемещение источника света |

### Вариант В (`c.c`)

Несколько объектов, 2 цветных источника света, зеркальная составляющая, тени на плоскость.

| Клавиша | Действие |
|---------|----------|
| 1 / 2 | Выбор активного источника света |
| ← / → / ↑ / ↓ | Перемещение выбранного источника |

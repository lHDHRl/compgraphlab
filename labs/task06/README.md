# Задание 6. Проволочная 3D-модель

## Описание

Построение каркасных (wireframe) 3D-моделей с возможностью вращения и выбором типа проекции.

## Теория

### Вращение вокруг оси Y

$$x' = x \cos\alpha + z \sin\alpha, \quad z' = -x \sin\alpha + z \cos\alpha$$

### Вращение вокруг оси X

$$y' = y \cos\alpha - z \sin\alpha, \quad z' = y \sin\alpha + z \cos\alpha$$

### Ортографическая проекция

Отбрасываем координату $z$:

$$s_x = c_x + x \cdot \text{scale}, \quad s_y = c_y - y \cdot \text{scale}$$

### Перспективная проекция

$$s_x = c_x + \frac{x \cdot \text{fov}}{z + d}, \quad s_y = c_y - \frac{y \cdot \text{fov}}{z + d}$$

### Примитивы

- **Куб** — 8 вершин $(\pm 1, \pm 1, \pm 1)$, 12 рёбер
- **Октаэдр** — 6 вершин $(\pm 1, 0, 0)$, $(0, \pm 1, 0)$, $(0, 0, \pm 1)$, 12 рёбер
- **Цилиндр/конус** — аппроксимация $N$-гранной призмой
- **Тор** — параметрическая поверхность:

$$x = (R + r \cos v) \cos u, \quad y = r \sin v, \quad z = (R + r \cos v) \sin u$$

где $R$ — большой радиус, $r$ — малый радиус, $u, v \in [0, 2\pi)$.

### Удаление невидимых линий

Z-буфер: грани сортируются от дальних к ближним и рисуются с закраской фоном.

## Алгоритм

1. Задать вершины и рёбра модели
2. Применить матрицу вращения ко всем вершинам
3. Спроецировать вершины на экран (ортографически или перспективно)
4. Нарисовать рёбра между спроецированными вершинами

## Варианты

### Вариант А (`a.c`)

Куб + октаэдр, ортографическая проекция, вращение вокруг оси Y.

| Клавиша | Действие |
|---------|----------|
| ← / → | Вращение вокруг Y |

### Вариант Б (`b.c`)

Цилиндр + конус + пирамида, перспективная проекция, вращение вокруг X и Y.

| Клавиша | Действие |
|---------|----------|
| ← / → | Вращение вокруг Y |
| ↑ / ↓ | Вращение вокруг X |

### Вариант В (`c.c`)

Тор, перспективная проекция с настраиваемым фокусным расстоянием, z-сортировка граней.

| Клавиша | Действие |
|---------|----------|
| ← / → | Вращение вокруг Y |
| ↑ / ↓ | Вращение вокруг X |
| + / − | Изменение фокусного расстояния |

# Задание 3. Фракталы (Рекурсия)

## Описание

Построение фрактальных структур: дерево Пифагора и L-системы.

## Теория

### Дерево Пифагора

На каждой стороне квадрата строится равнобедренный прямоугольный треугольник, на катетах которого строятся новые квадраты.

Масштаб каждого уровня:

$$s = \frac{1}{\sqrt{2}}$$

Углы ветвления: $\pm 45°$. После $n$ уровней размер квадрата:

$$a_n = a_0 \cdot \left(\frac{1}{\sqrt{2}}\right)^n$$

### L-системы

Формальная грамматика, задаваемая аксиомой и правилами подстановки.

Символы черепашьей графики:
- $F$ — движение вперёд с рисованием
- $+$ — поворот вправо на угол $\delta$
- $-$ — поворот влево на угол $\delta$
- $[$ — сохранить состояние (push)
- $]$ — восстановить состояние (pop)

**Куст:** аксиома $F$, правило:

$$F \to F[+F]F[-F]F, \quad \delta = 25.7°$$

**Папоротник:** аксиома $X$, правила:

$$X \to F+[[X]-X]-F[-FX]+X$$
$$F \to FF, \quad \delta = 25°$$

Длина шага на итерации $n$:

$$l_n = l_0 \cdot k^n, \quad k \approx 0.5$$

## Алгоритм работы

1. Задать начальные параметры (глубина, угол, длина)
2. Рекурсивно (или через подстановку строк для L-систем) построить фрактал
3. Отрисовать результат

## Управление

| Клавиша | Описание |
|---|---|
| `UP` / `DOWN` | Увеличить / уменьшить глубину рекурсии |
| `R` | Перегенерация со случайными параметрами (вариант Б) |
| `1` / `2` | Переключение куст / папоротник (вариант В) |
| `ESC` | Выход |

## Варианты

### Вариант А (`a.c`)
Классическое дерево Пифагора. Квадраты рисуются как прямоугольники, углы ветвления $\pm 45°$. Глубина регулируется клавишами `UP`/`DOWN`.

### Вариант Б (`b.c`)
Дерево со случайными углами ветвления в диапазоне $[30°, 60°]$. Рисуются только линии (скелет). Клавиша `R` — перегенерация с новыми случайными углами.

### Вариант В (`c.c`)
L-системы. Клавиши `1`/`2` переключают между кустом и папоротником. Листва отображается кружками на концах ветвей.

# Задание 11. Параметрические кривые

## Описание

Построение и редактирование параметрических кривых: Безье, B-Spline, Катмулл-Ром, NURBS. Контрольные точки задаются мышью, кривая перестраивается в реальном времени.

## Теория

### Кривая Безье

Кривая Безье степени $n$ по $n+1$ контрольным точкам:

$$B(t) = \sum_{i=0}^{n} C_n^i \cdot (1-t)^{n-i} \cdot t^i \cdot P_i, \quad t \in [0,1]$$

Для кубической кривой ($n=3$, 4 точки):

$$B(t) = (1-t)^3 P_0 + 3(1-t)^2 t \, P_1 + 3(1-t) t^2 P_2 + t^3 P_3$$

**Алгоритм де Кастельжо** — рекурсивная линейная интерполяция: на каждом шаге пары соседних точек заменяются их интерполяцией $P_i^{(k)} = (1-t) P_i^{(k-1)} + t \, P_{i+1}^{(k-1)}$, пока не останется одна точка.

### Uniform B-Spline (кубический)

Кривая определяется базисными функциями $N_{i,k}(t)$:

$$C(t) = \sum_{i} N_{i,k}(t) \cdot P_i$$

Кривая проходит **рядом** с контрольными точками (аппроксимация), обеспечивая $C^2$-гладкость.

### Катмулл-Ром

Интерполяционный сплайн — проходит **через** контрольные точки. Для сегмента между $P_i$ и $P_{i+1}$ используются четыре точки: $P_{i-1}, P_i, P_{i+1}, P_{i+2}$. Касательный вектор:

$$m_i = \frac{P_{i+1} - P_{i-1}}{2}$$

### NURBS

Рациональная кривая с весами $w_i$:

$$C(t) = \frac{\sum_{i=0}^{n} w_i \cdot N_{i,k}(t) \cdot P_i}{\sum_{i=0}^{n} w_i \cdot N_{i,k}(t)}$$

Вес $w_i$ управляет «притяжением» кривой к точке $P_i$: больше вес — кривая ближе к точке.

## Алгоритм

1. Хранить массив контрольных точек и их весов.
2. Для $t$ от 0 до 1 с шагом $\Delta t$ вычислить точку на кривой по соответствующей формуле.
3. Соединить вычисленные точки отрезками.
4. Отрисовать характеристическую ломаную (контрольный полигон).

## Варианты

| Вариант | Описание |
|---------|----------|
| **А** (`a.c`) | Кривая Безье по 4 точкам. Перетаскивание точек мышью. Отображение характеристической ломаной. |
| **Б** (`b.c`) | Переключение между Безье / B-Spline / Catmull-Rom по `TAB`. Правый клик — добавить точку, `D` — удалить ближайшую. |
| **В** (`c.c`) | NURBS с весами. Колёсико мыши изменяет вес выбранной точки. |

## Управление

- **ЛКМ** — выбрать / перетащить контрольную точку
- **ПКМ** — добавить точку (вариант Б)
- **D** — удалить ближайшую точку (вариант Б)
- **TAB** — переключить тип кривой (вариант Б)
- **Колёсико мыши** — изменить вес точки (вариант В)

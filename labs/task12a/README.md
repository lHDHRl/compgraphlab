# Задание 12А. Плавающий горизонт

## Описание

Визуализация трёхмерных поверхностей методом плавающего горизонта. Поверхность задаётся функцией $y = f(x, z)$ и отображается с возможностью вращения.

## Теория

### Алгоритм плавающего горизонта

Для каждого столбца экрана хранятся два значения:
- $y_{\min}[x]$ — верхний горизонт (минимальный $y$ экрана, т.к. ось вверх)
- $y_{\max}[x]$ — нижний горизонт

Поверхность рисуется **от дальних строк к ближним**. Для каждой точки поверхности с экранными координатами $(sx, sy)$:

- Если $sy < y_{\min}[sx]$ — точка выше верхнего горизонта → рисуем, обновляем $y_{\min}[sx] = sy$
- Если $sy > y_{\max}[sx]$ — точка ниже нижнего горизонта → рисуем, обновляем $y_{\max}[sx] = sy$
- Иначе — точка скрыта, пропускаем

### Проекция 3D → 2D

С учётом вращения на угол $\alpha$:

$$sx = cx + (x \cos\alpha - z \sin\alpha) \cdot s$$

$$sy = cy + (-y + z \cos\alpha \cdot 0.3 + x \sin\alpha \cdot 0.3) \cdot s$$

где $cx, cy$ — центр экрана, $s$ — масштаб.

## Алгоритм

1. Инициализировать массивы $y_{\min}[i] = +\infty$, $y_{\max}[i] = -\infty$ для каждого столбца.
2. Перебрать строки поверхности от дальней к ближней.
3. Для каждой точки строки вычислить экранные координаты $(sx, sy)$.
4. Сравнить $sy$ с $y_{\min}[sx]$ и $y_{\max}[sx]$, при необходимости нарисовать и обновить горизонт.

## Варианты

| Вариант | Описание |
|---------|----------|
| **А** (`a.c`) | Поверхность $y = \sin(x) \cdot \cos(z)$. Вращение клавишами `LEFT`/`RIGHT`. |
| **Б** (`b.c`) | Поверхность $y = \sin(x^2 + z^2)$ (кратер). Вращение `LEFT`/`RIGHT`. |
| **В** (`c.c`) | Две поверхности разными цветами. Вращение по двум осям: `LEFT`/`RIGHT` и `UP`/`DOWN`. |

## Управление

- **LEFT / RIGHT** — вращение вокруг вертикальной оси
- **UP / DOWN** — вращение вокруг горизонтальной оси (вариант В)

# Задание 2. Отсечение (Clipping)

## Описание

Реализация алгоритмов отсечения отрезков и полигонов прямоугольным окном.

## Теория

### Алгоритм Коэна-Сазерленда

Каждому концу отрезка присваивается 4-битный код области:

$$\text{code} = \text{LEFT} \cdot 1 + \text{RIGHT} \cdot 2 + \text{BOTTOM} \cdot 4 + \text{TOP} \cdot 8$$

- Бит `LEFT` ($1$): $x < x_{\min}$
- Бит `RIGHT` ($2$): $x > x_{\max}$
- Бит `BOTTOM` ($4$): $y < y_{\min}$
- Бит `TOP` ($8$): $y > y_{\max}$

Логика принятия решения для кодов $c_1$, $c_2$:

- $c_1 \,|\, c_2 = 0$ — отрезок полностью видим
- $c_1 \,\&\, c_2 \neq 0$ — отрезок полностью невидим
- Иначе — вычислить пересечение с границей окна и повторить

### Алгоритм Лианга-Барски

Параметрическое представление отрезка:

$$P(t) = P_1 + t(P_2 - P_1), \quad t \in [0, 1]$$

Для каждой границы окна вычисляется параметр $t$:

$$t = \frac{p_i}{q_i}, \quad \text{где } p_i, q_i \text{ — коэффициенты для } i\text{-й границы}$$

- $p_1 = -(x_2 - x_1)$, $q_1 = x_1 - x_{\min}$
- $p_2 = x_2 - x_1$, $q_2 = x_{\max} - x_1$
- $p_3 = -(y_2 - y_1)$, $q_3 = y_1 - y_{\min}$
- $p_4 = y_2 - y_1$, $q_4 = y_{\max} - y_1$

Найти пересечение интервалов $[t_{\min}, t_{\max}]$. Если $t_{\min} > t_{\max}$ — отрезок невидим.

### Алгоритм Сазерленда-Ходжмана (полигоны)

Последовательное отсечение полигона по каждой стороне окна. Для каждого ребра $(S, P)$ полигона и границы окна:

| $S$ внутри | $P$ внутри | Результат |
|---|---|---|
| Да | Да | Добавить $P$ |
| Да | Нет | Добавить пересечение |
| Нет | Да | Добавить пересечение и $P$ |
| Нет | Нет | Ничего |

## Алгоритм работы

1. Задать набор отрезков/полигонов и прямоугольное окно отсечения
2. Применить соответствующий алгоритм отсечения
3. Отобразить результат: видимые части — одним цветом, отсечённые — другим

## Управление

| Клавиша/действие | Описание |
|---|---|
| `LEFT` / `RIGHT` / `UP` / `DOWN` | Перемещение окна отсечения |
| `+` / `-` | Изменение размера окна |
| `ESC` | Выход |

## Варианты

### Вариант Б (`b.c`)
Прямоугольники, повёрнутые на случайный угол. Отсечение алгоритмом Лианга-Барски.

### Вариант В (`c.c`)
Хаотично расположенные прямоугольники. Заливка видимых частей полигонов алгоритмом Сазерленда-Ходжмана.

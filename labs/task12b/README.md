# Задание 12Б. Z-буфер

## Описание

Рендеринг трёхмерных объектов методом Z-буфера с растеризацией треугольников, освещением и текстурированием.

## Теория

### Z-буфер

Массив глубин $\text{zbuf}[W \times H]$, инициализированный значением $+\infty$. Для каждого пикселя $(x, y)$ растеризуемого треугольника:

$$\text{если } z < \text{zbuf}[x][y]: \quad \text{рисуем пиксель, } \text{zbuf}[x][y] = z$$

### Барицентрические координаты

Для точки $P$ внутри треугольника $ABC$:

$$P = u \cdot A + v \cdot B + w \cdot C, \quad u + v + w = 1$$

$$u = \frac{S(\triangle PBC)}{S(\triangle ABC)}, \quad v = \frac{S(\triangle APC)}{S(\triangle ABC)}, \quad w = 1 - u - v$$

Точка внутри треугольника, если $u, v, w \geq 0$. Интерполяция глубины:

$$z = u \cdot z_A + v \cdot z_B + w \cdot z_C$$

Аналогично интерполируются цвет, нормаль, UV-координаты.

### Освещение по Ламберту

$$I = \max(0, \, \vec{N} \cdot \vec{L})$$

где $\vec{N}$ — нормаль поверхности, $\vec{L}$ — направление к источнику света.

### Затенение по Гуро

Интенсивность $I$ вычисляется в вершинах треугольника, затем интерполируется по пикселям через барицентрические координаты.

### Текстурирование

UV-координаты интерполируются по треугольнику, цвет берётся из текстуры:

$$\text{color} = \text{texture}(u \cdot u_A + v \cdot u_B + w \cdot u_C, \; u \cdot v_A + v \cdot v_B + w \cdot v_C)$$

### Туман

$$\text{finalColor} = \text{lerp}(\text{pixelColor}, \, \text{fogColor}, \, \text{fogFactor})$$

$$\text{fogFactor} = \text{clamp}(z \cdot \text{density}, \, 0, \, 1)$$

## Алгоритм

1. Инициализировать Z-буфер значением $+\infty$.
2. Для каждого треугольника: найти ограничивающий прямоугольник на экране.
3. Для каждого пикселя в прямоугольнике: вычислить барицентрические координаты.
4. Если $u, v, w \geq 0$ — точка внутри: интерполировать $z$, проверить Z-буфер, вычислить цвет.

## Варианты

| Вариант | Описание |
|---------|----------|
| **А** (`a.c`) | Куб + сфера. Плоская закраска (Lambert). Вращение `LEFT`/`RIGHT`. |
| **Б** (`b.c`) | Пирамида + цилиндр + тор. Затенение по Гуро. Вращение стрелками. |
| **В** (`c.c`) | Текстурированный куб (шахматная текстура) + туман. `+`/`-` — плотность тумана. |

## Управление

- **LEFT / RIGHT** — вращение объектов
- **UP / DOWN** — дополнительное вращение (вариант Б)
- **+ / −** — изменение плотности тумана (вариант В)
